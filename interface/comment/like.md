# /comment/{commentId}/like
评论的点赞接口

## POST
点赞评论

### 参数说明
无

#### 上传数据
无

#### 返回数据
```
{}
```

#### 补充说明
1. 需要登录
2. 评论点赞数 + 1

#### 错误处理
* 304：已经点赞
* 400：参数错误
* 401：未登录
* 404：评论不存在

## DELETE
取消点赞

### 参数说明
无

#### 上传数据
无

#### 返回数据
```
{}
```

#### 补充说明
1. 需要登录
2. 评论点赞数 - 1

#### 错误处理
* 401：未登录
* 404：评论不存在
